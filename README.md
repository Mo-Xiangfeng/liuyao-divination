# 六爻卜卦网站

一个基于易经六爻的在线占卜网站，集成DeepSeek AI智能解读功能。

## 通过Netlify部署，DeepSeek密钥请自行配置

## 功能特点

- 🎯 六爻卦象生成
- 🤖 AI智能卦辞解读
- 📱 响应式设计
- 💾 本地历史记录
- 🎨 传统风格界面

## 技术栈

- HTML5
- CSS3
- JavaScript
- DeepSeek API

## 使用方法

1. 点击"开始占卜"按钮
2. 静心思考您的问题
3. 查看生成的卦象和解读
4. 历史记录自动保存

## 部署

直接打开 `index.html` 文件或在任何静态服务器上部署。

## 注意事项

本工具仅供娱乐参考，请理性看待占卜结果。
